# Immersive Themes (沉浸式氛圍體驗)

這是一個專注於提供極致視聽體驗的 Web 應用程式，旨在透過細膩的視覺特效與客製化的白噪音聲景，幫助使用者放鬆身心或專注工作。

## ✨ 特色功能 (Features)

### 1. 三大沉浸主題
*   **午夜海岬 (Deep Night)**：
    *   **視覺**：高擬真的 SVG 海浪動畫、佇立的燈塔、閃爍的星空與流動的雲霧。
    *   **聽覺**：雨聲、海浪聲、風聲。
    *   **氛圍**：鎮靜、降溫，適合深層思考。
*   **迷霧森林 (Misty Forest)**：
    *   **視覺**：多層次森林剪影、穿透樹梢的耶穌光 (God Rays)、飄落的樹葉與螢火蟲。
    *   **聽覺**：林間風聲、蟲鳴鳥叫、潺潺溪流。
    *   **氛圍**：療癒、有機，適合閱讀與放鬆。
*   **溫暖營火 (Warm Ember)**：
    *   **視覺**：帳篷剪影、燃燒的營火光暈、飄散的火星與熱氣擾動效果。
    *   **聽覺**：營火劈啪聲、落葉沙沙聲、夜蟲低鳴。
    *   **氛圍**：溫暖、安全感，適合冥想與休憩。

### 2. 動態聲景控制 (Dynamic Soundscapes)
*   每個主題擁有獨立的音軌組合。
*   使用者可透過滑桿獨立調整每種環境音的音量，打造專屬的聽覺體驗。
*   支援自動循環播放與平滑切換。

### 3. 精緻的視覺特效
*   **粒子系統**：使用原生 JavaScript 實現高效能的粒子效果（雨滴、星星、螢火蟲、火星）。
*   **SVG 動畫**：高解析度的海浪與背景元素，結合 CSS Animation 實現有機的自然律動。
*   **玻璃擬態 UI**：現代化的介面設計，具備模糊背景與細緻的光影質感。
*   **歡迎視窗**：帶有星空紋理與微光文字的引導頁面，提供沉浸式的入場儀式感。

### 4. 貼心互動設計
*   **自動摺疊面板**：閒置 20 秒後自動收起控制面板，還給使用者純淨的視覺空間。
*   **響應式設計 (RWD)**：完美支援桌面與行動裝置瀏覽。

## 🛠️ 技術堆疊 (Tech Stack)

*   **HTML5**：語意化標籤結構。
*   **CSS3**：
    *   CSS Variables (變數) 管理主題色系。
    *   Keyframe Animation 實現複雜動畫。
    *   Flexbox & Grid 佈局。
*   **Vanilla JavaScript (ES6+)**：
    *   無任何第三方依賴 (No Frameworks)。
    *   模組化的程式碼結構。
    *   Web Audio API 整合。

## 🚀 如何執行 (How to Run)

1.  下載或 Clone 此專案。
2.  直接使用瀏覽器開啟 `index.html` 檔案即可開始體驗。
3.  建議佩戴耳機以獲得最佳沉浸效果。
